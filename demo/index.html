GitHub Corners

Phew, GitHub is over seven years old now... and is unquestionably synonomous with open source. After 7 years, those GitHub ribbons are more than overdue for a cleaner, more modern alternative. This is my take.

By using SVG, these corners can be clean and sharp. Any color, and size. As well as have a little character... on hover for desktop, and on page load for mobile devices.

You can build get the code here... or edit the svg's however you like!

Customize Appearance

GitHub Corners follows the BEM naming methodology. Use these classes in your stylesheet to customize GitHub Corner's appearance.

.gh-corner__octocat
Height, Width (default 80px)
.gh-corner__octocat__mask
Fill (default #000)
.gh-corner__octocat__body
Fill (default #000)
.gh-corner__octocat__tail
Fill (default #000)

Positioning

Combine these modifier classes with .gh-corner to change it's positioning.

.gh-corner
Default positioning (top-right corner of window)
.gh-corner--right
Position on right side of window
.gh-corner--left
Position on left side of window
.gh-corner--fixed
Make corner "sticky" in window
